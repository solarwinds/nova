These colors should not be directly used in implemented components! Instead, use the colors in the "Semantic" tab.

The reason there are 2 different palettes is that this palette defines every color that is used. The semantic palette described HOW the colors are used.

Breaking it up like this allows us to change every instance of a color, or a small subset of a color (i.e change all blue borders to purple).
