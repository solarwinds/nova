bits schematics

All schematics work in similar way. To use them, you can call it by first doing ng add @nova-ui/bits then ng g @nova-ui/bits:SCHEMATICNAME and add option, available for each schematic. Note: for most of the schematics, name can be set without specifying option, just type it as the first argument in command line.
