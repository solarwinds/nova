The Table schematic provides the ability to create a generic table that can cover a majority of use cases. Its main responsibility is to integrate the functionality of the nui-table component with other components commonly used with a table such as nui-search and nui-paginator.

Notable Command Line Options

Basic Usage

This example illustrates the output of the CLI command without any modifications. Consumers are able to build upon the generated boilerplate to provide additional functionality depending on their use case. Data manipulations (filtering/pagination/sorting/search) are handled by the ClientSideDataSource service, which can be extended with custom filtering behavior.

ng g @nova-ui/bits:table --p=app --name=basic-table --enable-search=false --enable-sort=false

Table Sorting

ng g @nova-ui/bits:table --p=app --name=table-with-sort --enable-sort --enable-search=false

Table with Search

ng g @nova-ui/bits:table --p=app --name=table-with-search --enable-search --enable-sort=false

Pagination

This example shows how to add pagination to nui-table with a remote data source.

ng g @nova-ui/bits:table --name=table-with-pagination --p=app --pagingMode=pagination --dataSource=serverSide

Selection

This example shows how to add selection to nui-table with a remote data source.

ng g @nova-ui/bits:table --name=table-with-pagination --p=app --pagingMode=pagination --dataSource=serverSide --selectionMode=multi

Virtual Scroll

This example shows how to add virtual scroll to nui-repeat with a remote data source.

Since the example makes use of search and sorting filters, it's important to note that the viewport manager's paging progress must be reset each time the data is refreshed with new filtered results whenever the filters are changed. ng g @nova-ui/bits:table --name=table-with-virtual-scroll --p=app --pagingMode=virtualScroll --dataSource=serverSide

Virtual Scroll with Custom Strategy

ng g @nova-ui/bits:table --name=table-with-virtual-scroll --p=app --pagingMode=virtualScroll --dataSource=serverSide --virtualScrollStrategy=custom
