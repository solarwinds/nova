The Filtered View schematic provides the ability to create a generic filtered view that can cover many use cases for filtering a set of data. Its main responsibility is to integrate a filter group panel with a list or table view that displays the filtered data.

Notable Command Line Options

This string option allows you to select how the filtered data will be presented to the user. The values currently supported for this option are table and list. The default value is list.

Filtered View with List

This example shows the output of the CLI command without any modifications.

ng g @nova-ui/bits:filtered-view --p=app --name=filtered-view-with-list --presentationType=list

List with Pagination

ng g @nova-ui/bits:filtered-view --p=app --name=filtered-view-list-with-pagination --presentationType=list --dataSource=serverSide --pagingMode=pagination

List with Virtual Scroll

ng g @nova-ui/bits:filtered-view --p=app --name=filtered-view-virtual-scroll --presentationType=list --dataSource=serverSide --pagingMode=virtualScroll

Filtered View with Table

This example shows the output of the CLI command shown below.

ng g @nova-ui/bits:filtered-view --name=filtered-view-with-table --p=app --presentationType=table

Table with Pagination

ng g @nova-ui/bits:filtered-view --p=app --name=filtered-view-table-with-pagination --presentationType=table --dataSource=serverSide --pagingMode=pagination

Table with Selection

ng g @nova-ui/bits:filtered-view --p=app --name=filtered-view-table-with-selection --presentationType=table --dataSource=serverSide --pagingMode=pagination

Table with Virtual Scroll

ng g @nova-ui/bits:filtered-view --p=app --name=filtered-view-table-with-virtual-scroll --presentationType=table --dataSource=serverSide --pagingMode=virtualScroll

Table with Virtual Scroll and Selection

ng g @nova-ui/bits:filtered-view --p=app --name=filtered-view-table-with-virtual-scroll --presentationType=table --dataSource=serverSide --pagingMode=virtualScroll

Table with Custom Virtual Scroll

ng g @nova-ui/bits:filtered-view --p=app --name=filtered-view-table-with-virtual-scroll --presentationType=table --dataSource=serverSide --pagingMode=virtualScroll --virtualScrollStrategy=custom
