Required Modules

  • NuiCheckboxModule

Basic Usage

The <nui-checkbox-group> component is used for combining <nui-checkbox> components into a group. Use the name input to set the "name" attribute for the inner <nui-checkbox> input elements. To get the values of the <nui-checkbox> child components use the values input which is an array of checked state values. To act on checkbox value changes, bind to the (valuesChange) output.

Inline

Use the nui-checkbox-group-inline class to display checkbox group inline.

Part of Form

<nui-checkbox-group> can be used in reactive forms.

Justified

To justify <nui-checkbox> to the width of <nui-checkbox-group> (or simply parent element, in case <nui-checkbox> is used not as a part of <nui-checkbox-group>), add nui-checkbox--justified class to the <nui-checkbox>.
