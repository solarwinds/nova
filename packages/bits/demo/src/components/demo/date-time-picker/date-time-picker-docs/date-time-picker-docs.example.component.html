Required Modules

  • NuiDateTimePickerModule

Basic Usage

<nui-date-time-picker> allows user to select a specific time moment setting correspondent date and time. For time selecting nui-time-picker is used, for date selecting nui-date-picker.

Empty Initial State

<nui-date-time-picker> allows user to init a component with the empty values using the attribute "initEmpty". For time selecting nui-time-picker is used, for date selecting nui-date-picker.

Reactive Form Usage

<nui-date-time-picker> may be used in a reactive form by setting the formControl input.

Min/Max value

To set min or max date for date field of dateTimePicker use minDate or maxDate input (as shown in DatePicker docs).

Disabled Mode

To disable dateTimePicker set isDisabled input to true

Inline Mode

Set displayMode mode binding to 'inline' in order to set row direction for dateTimePicker's input fields.

Timezones

As far as DateTimePickerComponent uses DatePickerComponent it can also be used with handling timezones. Setting handleTimezone flag to true makes handling timezone.

Appending Popup Controls to the Document Body

DateTimePicker popups (for both DatePicker and TimePicker) can be appended to the document body by setting the appendToBody input to true.
