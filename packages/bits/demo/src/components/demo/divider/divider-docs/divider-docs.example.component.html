Basics

Nui-divider can be used as a horizontal or vertical line divider with customizable margins.

Horizontal Dividers

size binding corresponds to divider's margins. There are 4 sizes for divider:

  • no-margin - margin-top and margin-bottom equals 0
  • extra-small - margin-top and margin-bottom equals 5px
  • small - margin-top and margin-bottom equals 10px
  • medium - margin-top and margin-bottom equals 15px. This is the default.

Default margin for horizontal divider is medium.

Vertical Dividers

Vertical divider could be added by using [isVertical]="true" binding on divider.

As well as horizontal dividers, vertical has the same sizes so it affects left and right margins respectively.

Default margin for vertical divider is extra-small.

NOTE: divider should be placed into flex container (only applies to vertical dividers).
