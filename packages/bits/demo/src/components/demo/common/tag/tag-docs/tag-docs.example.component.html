Basic Usage

To show tag add .nui-tag class to the element.

Statuses Usage

To change tag styles for specific statuses you can use appropriate classes for that (for example, .nui-tag-status). As default, text color is black. You can change it by adding .nui-tag-text-light class.

Statuses Usage With Border

To change tag styles for specific statuses you can use appropriate classes for that (for example, .nui-tag-with-border-status).
