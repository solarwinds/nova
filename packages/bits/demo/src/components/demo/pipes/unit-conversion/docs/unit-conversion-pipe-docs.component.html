The unitConversion pipe can be used to convert a large value of a small base unit to a smaller value of a larger unit--for example, converting 1024 bytes to 1 kilobyte. The converted unit label is automatically appended to the output. Supported units are of type UnitOption and include the following: "generic" (e.g. 1k for 1000), "bytes", "bytesPerSecond", "bitsPerSecond", and "hertz".

Required Modules

  • NuiCommonModule

Basic Usage

To get started using the pipe, take a look at the template source of the example below.

Parameters

The pipe accepts the following parameters:

  • value - The number to be converted.
  • precision - Precision of the formatted value. Extra trailing zeros are removed independently of the precision.
  • plusSign - If true and the source value is greater than zero, the output is prefixed with a plus sign.
  • unit - The base unit of the conversion. Accepts arguments of type UnitOption.
