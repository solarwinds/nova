Core IT

.nui-text-product
This style uses Roboto font, however, Nova Bits styles do not include it. You can easily add it in following ways:
  1. Import font-roboto.less directly like @import "@nova-ui/bits/sdk/less/mixins/font-roboto.less"
  2. Use .font-roboto() mixin to load Roboto font
  3. Mixin Example:

    .font-roboto();
    .css-class {

    font-family: @font-family-roboto;

    }

  1. Find the compiled stylesheet file under the @nova-ui/bits/bundles/css/styles-font-roboto.css
  2. Add it to angular.json to set Roboto font globally, or use it locally by importing the file into the stylesheet where this font is needed.
  3. Angular.json Example:

    "styles": [

    "[path_to_node_modules]/@nova-ui/bits/bundles/css/styles-font-roboto.css"

    ],
