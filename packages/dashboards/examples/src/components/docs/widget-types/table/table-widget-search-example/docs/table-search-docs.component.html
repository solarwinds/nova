Table Widget with Search Functionality

Table Widget can have search functionality. This page contains information only about setting up the search, so before proceeding get familiar with the following:

  • Data Sources - for information about Data Sources and their Features.
  • Table Widget - for configuring a table widget itself.

Configuring the Data Source

To use search within the Data Source you need to use data source Features. Data Source should implement IDataSource interface and have features property set.
