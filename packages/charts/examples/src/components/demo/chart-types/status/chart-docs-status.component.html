Status Charts

A Status Chart represents the state of a system over time.

Basic Usage

For the process of creating a status chart, take a look at the inline comments explaining it in the example source below.

Please note that InteractionLinePlugin and InteractionLabelPlugin plugins should be added manually.

Legend Interaction

If there is a space for legend, it is possible to communicate current status not via tooltip, but via legend.

Please note how this is implemented using chartAssist.legendLabelData$ stream.
