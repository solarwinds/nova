Gauges

Gauges can be an effective way to display a single metric value in visual form. And, Nova Charts provides the ability to create three types of gauges with the assistance of a utility class called GaugeUtil. The primary functions provided by the GaugeUtil that get you most of the way towards a functioning gauge are:

  • assembleSeriesSet: assembles a gauge series set with all of the standard requisite scales, renderers, accessors, etc. needed for creating a gauge visualization based on the provided IGaugeConfig object and GaugeMode.
  • updateSeriesSet: updates a gauge series set with the latest value, max value, thresholds, and color accessors based on the provided IGaugeConfig object.

Donut Type

A donut gauge is well suited for use cases that consist of a larger area to display a metric visualization such as a dashboard. For use cases, with a more compact display area, consider the linear gauge type also documented on this page.

Basic Donut Usage

From the Nova perspective, a donut gauge is basically the same as a donut chart but with data series formatted in a way that will result in a gauge visualization. To create a donut gauge, first instantiate a donut chart, and then update the chart with a series set generated by the GaugeUtil's assembleSeriesSet function. It takes an IGaugeConfig object containing a value and a max value for the gauge as the first argument and a GaugeMode value of GaugeMode.Donut for the second argument. Later, you may optionally update the gauge with a new value, max value, thresholds, or even a different color accessor using the GaugeUtil's updateSeriesSet function.

Adding Donut Thresholds and Threshold Markers

The next example adds thresholds and their associated colors, markers, and labels. To see the labels, hover your mouse over the arcs of the donut.

To add thresholds, specify an array of ordered numbers on the thresholds property of the IGaugeConfig passed to the GaugeUtil's assembleSeriesSet function. Each number corresponds to a threshold. For instance [25, 75] indicates a warning threshold of 25 and a critical threshold of 75. By default, Nova gauges provide a standard color accessor for the thresholds which outputs a yellow color for warning and a red color for critical. The default color accessor can be overridden to handle any number of thresholds by setting a custom DataAccessor value for the IGaugeConfig object's quantityColorAccessor property.

To enable the threshold marker dots, set the enableThresholdMarkers property on the IGaugeConfig object to true. And, to add labels for the markers, add a DonutGaugeLabelsPlugin instance to the chart.
